# Guia Rápido - Lifebooks

Este guia apresenta as funcionalidades essenciais do Lifebooks para você começar rapidamente.

## Navegação Básica

### Usando o Menu Lateral

O menu lateral organiza toda a documentação em categorias:
- **Guias**: Tutoriais passo a passo
- **FAQ**: Perguntas frequentes
- **Referência**: Documentação técnica detalhada

### Busca Rápida

Pressione `Ctrl+K` (ou `Cmd+K` no Mac) para abrir a busca rápida e encontrar qualquer conteúdo.

## Tarefas Comuns

### Como Criar um Novo Documento

1. Acesse a seção de documentos
2. Clique em "Novo Documento"
3. Preencha os campos necessários
4. Salve e publique

### Como Organizar Conteúdo

Use as tags e categorias para manter seu conteúdo organizado:
- Tags ajudam na busca
- Categorias estruturam a navegação
- Módulos separam diferentes áreas

## Dicas e Truques

💡 **Dica**: Use atalhos de teclado para navegar mais rapidamente pela documentação.

⚡ **Performance**: Utilize a busca para encontrar informações rapidamente em vez de navegar manualmente.

🎯 **Organização**: Sempre adicione tags relevantes aos seus documentos.
