# FAQ - Lifebooks

Perguntas frequentes sobre o módulo Lifebooks.

## Perguntas Gerais

### O que é o Lifebooks?

O Lifebooks é um módulo de documentação e help desk que permite gerenciar e acessar informações de forma organizada e eficiente.

### Como faço para começar?

Consulte nosso [Guia Rápido](./guia-rapido.md) para instruções detalhadas sobre como começar.

### Onde encontro mais informações?

Navegue pelas diferentes seções no menu lateral ou use a busca para encontrar tópicos específicos.

## Problemas Técnicos

### A página não carrega corretamente

Experimente:
1. Limpar o cache do navegador
2. Recarregar a página (F5 ou Ctrl+R)
3. Verificar sua conexão com a internet

### Não consigo encontrar um documento

Use a busca (Ctrl+K) para localizar documentos específicos. Certifique-se de usar palavras-chave relevantes.

### Como reporto um problema?

Entre em contato com o suporte através dos canais oficiais listados na seção de contato.

## Recursos e Funcionalidades

### Posso exportar documentos?

Sim, use a função de exportação disponível em cada documento para salvá-lo em diferentes formatos.

### Como compartilho um documento?

Copie o link do documento e compartilhe com outros usuários. Certifique-se de que eles tenham as permissões necessárias.

### O sistema suporta múltiplos idiomas?

Sim, o sistema está configurado para suportar múltiplos idiomas, com português (pt-BR) como idioma padrão.
