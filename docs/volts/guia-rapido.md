# Guia Rápido - Volts

Instruções essenciais para começar a usar o módulo Volts rapidamente.

## Configuração Inicial

### Conectando Dispositivos

1. Acesse o painel de dispositivos
2. Clique em "Adicionar Dispositivo"
3. Siga o assistente de configuração
4. Teste a conexão

### Configurando Monitoramento

Configure os parâmetros de monitoramento:
- Intervalos de medição
- Limites de alerta
- Tipos de dados coletados
- Frequência de relatórios

## Monitoramento e Análise

### Dashboard Principal

O dashboard oferece visão geral:
- Consumo atual de energia
- Histórico de uso
- Alertas ativos
- Status dos dispositivos

### Relatórios

Gere relatórios personalizados:
- Por período
- Por dispositivo
- Por tipo de evento
- Análises comparativas

## Manutenção e Alertas

### Configurando Alertas

Defina alertas para eventos críticos:
1. Acesse Configurações > Alertas
2. Defina condições de acionamento
3. Configure destinatários
4. Teste os alertas

### Gestão de Manutenção

Planeje e acompanhe manutenções:
- Manutenção preventiva
- Histórico de intervenções
- Agendamento de tarefas
- Registro de ocorrências

## Dicas de Uso

⚡ **Performance**: Configure intervalos de medição apropriados para balancear precisão e performance.

📊 **Análise**: Use os relatórios comparativos para identificar padrões de consumo.

🔔 **Alertas**: Configure alertas redundantes para eventos críticos.
