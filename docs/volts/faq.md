# FAQ - Volts

Perguntas frequentes sobre o módulo Volts.

## Configuração e Instalação

### Quais dispositivos são compatíveis?

O módulo Volts é compatível com diversos tipos de sensores e medidores de energia. Consulte a lista completa na documentação técnica.

### Como adiciono um novo dispositivo?

Acesse o painel de dispositivos e use o assistente de configuração para adicionar novos equipamentos.

### Preciso de hardware especial?

Depende do tipo de monitoramento desejado. Alguns recursos funcionam com hardware padrão, enquanto outros podem requerer equipamentos específicos.

## Monitoramento e Dados

### Com que frequência os dados são atualizados?

A frequência de atualização é configurável, variando de tempo real até intervalos mais espaçados conforme sua necessidade.

### Os dados históricos ficam armazenados por quanto tempo?

Por padrão, mantemos dados históricos por 12 meses, com opção de extensão mediante configuração.

### Posso exportar os dados?

Sim, todos os dados podem ser exportados em diversos formatos (CSV, Excel, PDF) para análise externa.

## Alertas e Notificações

### Como configuro alertas de emergência?

Acesse Configurações > Alertas e defina as condições críticas que devem acionar notificações imediatas.

### Posso receber alertas por SMS ou email?

Sim, o sistema suporta múltiplos canais de notificação incluindo SMS, email e push notifications.

### Os alertas funcionam 24/7?

Sim, o sistema de alertas opera continuamente para garantir que você seja notificado de eventos críticos a qualquer momento.

## Manutenção e Suporte

### Como solicito manutenção preventiva?

Use o módulo de gestão de manutenção para agendar intervenções preventivas e acompanhar o histórico.

### O que fazer em caso de falha do sistema?

Consulte o guia de troubleshooting ou entre em contato imediatamente com o suporte técnico para casos urgentes.

### Há treinamento disponível?

Sim, oferecemos treinamentos para uso avançado do sistema. Entre em contato para mais informações.

## Performance e Otimização

### Como otimizo o desempenho do sistema?

Ajuste os intervalos de coleta de dados e configure adequadamente os filtros nos dashboards.

### O sistema funciona offline?

Alguns recursos básicos funcionam offline, mas o monitoramento completo requer conexão com a internet.
